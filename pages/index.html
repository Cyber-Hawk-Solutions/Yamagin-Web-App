Jekyll PWA Starter

A Jekyll/PWA starter project. Find out more about the Jekyll PWA Stater on GitHub.

This is a boilerplate project to start writing with Jekyll. It can be published with your own host or directly on GitHub. You can also use other free hosting services like Netlify, CloudFlare or CloudCannon.

This Jekyll boilerplate includes:

  • Critical CSS
  • Service Worker
  • Web App manifest
  • RSS feed
  • IndieWeb microformats
